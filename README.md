# 🚀 Fullstack Dev Guide

An interactive and beginner-friendly developer reference covering the most important concepts in modern fullstack development.

Perfect for:

* 📚 Students learning web development
* 💻 Beginners exploring fullstack concepts
* 🔁 Quick developer reference
* 🧠 Revision before interviews
* 🛠️ Practicing frontend architecture without frameworks

---

# ✨ Features

* 🔍 Instant live search across all concepts
* 🏷️ Category-based filtering
* 📖 Multi-tab learning interface
* ✅ Mark concepts as learned
* 📊 Visual progress tracking
* 🔗 Related concepts navigation
* ⬅️➡️ Previous / next topic navigation
* 🎨 Clean responsive UI
* ⚡ Zero dependencies

# ⚙️ CI/CD Deployment Guide

This project supports automatic deployment using:

* GitHub Actions
* GitHub Pages
* CI/CD workflow

Whenever code is pushed to GitHub, the website automatically redeploys.

---

## 🚀 Push Updates

Whenever you update code:

```bash
git add .
git commit -m "Updated project"
git push
```

GitHub automatically redeploys your website.
